# Jumpbox

The Docker image contains a suite of CLI tools to help diagnose and debug issues running inside Kubernetes.

### Networking

```
nmap
telnet
ping
traceroute
nc / netcat
curl
```

### Kubernetes

```
kubectl
helm
docker
```

### GraphQL

```
graphqurl
```

### PostgreSQL (client)

```
psql
```

### Mongo (client)

```
mongo
```

### NodeJS

```
node
npm
```
